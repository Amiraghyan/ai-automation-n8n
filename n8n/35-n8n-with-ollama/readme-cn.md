## 🎯 目标

* 🚀 本地环境（Mac/Windows）快速安装和设置 Ollama，并结合 n8n 自动化流程。

* 🛠️ 掌握 WebUI 与 Docker 的部署技巧，完成与 Ollama API/本地模型的串接。
